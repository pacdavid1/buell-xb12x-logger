
# Buell XB12X Logger – Overview

Este proyecto convierte una Raspberry Pi en un dispositivo dedicado
para logging y visualización de datos DDFI-2 en una Buell XB12X.

## Objetivos
- Appliance headless
- Instalación en SD limpia
- Persistente tras reboot
- Acceso vía WiFi hotspot
- Dashboard web local

## No objetivos
- Acceso a Internet
- Uso general como Raspberry Pi
- Configuración manual post-install
