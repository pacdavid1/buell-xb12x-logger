
# Buell XB12X Logger – Overview

Este proyecto convierte una Raspberry Pi en un dispositivo dedicado
para logging y visualización de datos DDFI-2 en una Buell XB12X.

## Objetivos
- Appliance headless
- Instalación en SD limpia
- Persistente tras reboot
- Acceso vía WiFi hotspot
- Dashboard web local

## No objetivos
- Acceso a Internet
- Uso general como Raspberry Pi
- Configuración manual post-install

## Uso de Copilot

Copilot se utiliza como herramienta de apoyo para:
- análisis
- generación de ideas
- redacción de código y documentación

Las decisiones técnicas finales y la documentación oficial
se registran siempre en el repositorio.

Copilot no es considerado fuente de verdad del proyecto.
