# Registro de Decisiones Técnicas

## 2026-03-18 – Hotspot como infraestructura del sistema

Decisión:
El hotspot se maneja con NetworkManager, no desde Python.

Motivo:
Persistencia, robustez y simplicidad.

Estado:
Implementado y estable.

## 2026-03-18 – Documentación versionada como fuente de verdad

Decisión:
La documentación del proyecto vive dentro del repositorio,
en la carpeta /docs, y se versiona junto con el código.

Motivo:
Evitar pérdida de contexto en chats y asegurar trazabilidad
técnica a largo plazo.

Estado:
Implementado.
